MChat Help

MChat is a simple program that allows two people connected to an Ethernet network to chat with each other.
Among its main features is the fact that conversations are direct and leave no trace.
Each message sent is encrypted and then decrypted upon arrival using a special key.
However this mechanism cannot avoid other control methods such as software that
monitor the typing of keys.

Use

Computers that need to come into contact must be on the same network.
The work directory is C:\MChat there the program write all preferences but not chat discussions.
You can set your server and users preferences with preferences-panel.
Server and PC names are case sensitive.
The linking mechanism is as follows:

  • One of the two interlocutors must listen, so he must act only on the "Server" connection button. In server mode the program waits for a call
  • The other party must select from the list the name of the server and the name of PC that is listening and press the "Call" button
  • At this point, once the connection is established, the interlocutors can chat between them
  • Whoever acts as a server can open multiple instances of MChat and, by activating different servers, communicate with multiple clients. The conversation always remains one to one.
It is possible to cancel the chat either through the relative button or with the F1 function key.
Warning! This cleaning acts only on one's "blackboard" and not on that of the interlocutor.
Is is possible to hide MChat into systray with F2 function key.
